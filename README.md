# nodejs-web-vite-yarn-react-typescript-template

## Description
A react template for the vite packager.

## Tech stack
- yarn
- vite
- react
- typescript

## Docker stack
- node

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
